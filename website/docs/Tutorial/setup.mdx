## Unpack

Open a terminal and unpack the download package to your desired destination.
Make sure that your current user that read, write and execution permission in that folder.  
```
tar -xvf devity_onboarding_package.tar.gz
``` 
  
In the package you just unpacked are three sperate software packages.
The package should look like this.
```
.
├── cert_management
│   ├── cert_man
│   ├── config
│   └── creds
├── conf_management
│   ├── bin
│   │   └── conf_man
│   ├── conf.d
│   └── templates
├── initial
│   ├── data
│   ├── linux-client
└── mqtt_client
    └── demo-client
```
  
- Initial onboarding
  - Recieves initial data and configurations
- Configuration management
  - Generates and updates configurations for other apllications (e.g. certificate management)
- Certificate management
  - Generates new certificates and renews them when expiring

Now proceed to configurating your devie.
