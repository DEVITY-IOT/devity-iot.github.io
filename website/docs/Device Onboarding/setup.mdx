## Unpack

Open a terminal and unpack the download package to your desired destination.
Make sure that your current user have the read, write and execution permission in that folder.  
```
tar -xvf demo-kit.tar.gz
``` 
  
In the package that you just unpacked are three sperate software packages.
The package should look like this.
```
.
├── cert-man
│   ├── creds (with certificate management server)
│   ├── cert-man (executable)
│   └── config (configurations for certificate management)
├── conf-man
│   ├── conf.d
│   ├── conf-man
│   └── templates
├── mqtt-client
│   └── demo-client
├── linux-client
├── data
└── truststore

```

- Onboarding
  - Receives initial data and configurations
- Certificate management
  - Generates new certificates and renews them when expiring
- Configuration management
  - Generates and updates configurations for other applications (e.g. certificate management)
- truststore
  - Stores certificate authority certificates

Now proceed to configurating your devie.
